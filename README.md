# 中国互联网行业外商独资企业（WFOE）信息汇编

本项目系统梳理了在中国大陆活跃运营的外资互联网企业（WFOE）的工商信息、商业模式与法规环境，共收录 **34 家活跃 WFOE** 及 **10 家已退出/受限企业** 的对比参考。

## 数据范围

- **主要范围**：2016–2026 年间仍活跃运营的外资互联网 WFOE
- **历史追溯**：部分企业对比信息追溯至 1990 年代首批 WFOE（如微软 1995、甲骨文 1991）
- **核实提示**：所有工商信息均来源于公开渠道，建议通过[国家企业信用信息公示系统](https://www.gsxt.gov.cn)进行最终核实

## WFOE 子类型

报告涵盖《外商投资法》统一调整下的两种 WFOE 登记类型：

| 类型 | 工商登记表述 | 典型架构 |
|------|-------------|---------|
| **外国法人独资** | 有限责任公司（外国法人独资） | 母公司（美国/欧洲/日韩等）→ 中国大陆 WFOE |
| **台港澳法人独资** | 有限责任公司（台港澳法人独资） | 母公司 → 香港/台湾/澳门子公司 → 中国大陆 WFOE |
